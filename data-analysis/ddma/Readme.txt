Start this demo by running the file - demo.m. 
The demo generates a three-arm spiral data set of 3 dimensions, and then
 reduce the original data space to a 2-dimensional space. 

File descriptions: 

demo.m 
Demo for Discriminant Diffusion Maps Analysis (DDMA), a manifold learning
method for dimensionality reduction. 

ddma.m 
Main function of the discriminant diffusion maps analysis (DDMA)

gen_spir_3d.m
Generate a data set of a three-arm spiral
